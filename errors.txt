API ERRORS:
    DB:
    - DB-01: An error occurred while setting the data in the database
    - DB-02: An error occurred while saving the data in the database
    USER:
    - US-01: User not found
    - US-02: User already exists
    - US-03: User not logged in
    - US-04: User not authorized
    CONVERSION:
    - CV-01: Conversion not found
    - CV-02: Limit conversion reached (the user has reached the limit of conversions per plan, and can't create more)
    - CV-03: To foreing currency not found
    - CV-04: From foreing currency not found
    FOREING:
    - FG-01: Foreing currency not found
    - FG-02: Foreing code already exists
